
Endpoints *PRODUCTO*

| METHOD      | Endpoint     |Descripcion                       |    
|----------   |---------     |----------                        |
| GET         | /productos   |Muestra  todos los productos      |
| POST        | /productos   | crea un producto                 |
| PATCH       | /productos/id| modifica el precio de un producto|
| DELETE      | /productos/id| elimina un producto              |
